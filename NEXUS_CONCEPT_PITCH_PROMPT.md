# Prompt: Create a Concept Pitch Deck for Nexus

> **Instructions for Claude Desktop**: Create a **visually stunning, animation-rich PowerPoint presentation** (6 slides) that pitches the *concept* of **Nexus** — a team AI assistant for Azure cloud architects and engineers. This deck is designed to win stakeholder approval. Nothing has been built yet — sell the vision.
>
> **Tone**: Bold. Visionary. Confident. No jargon walls. Every slide should make someone lean forward.
>
> **Animation style**: Heavy use of entrance animations, morph transitions between slides, sequential reveals (text and elements appear one by one, not all at once). Think Apple keynote meets Microsoft Build.
>
> **Design language**: Dark mode. Cinematic. Premium tech aesthetic.
> - Background: Deep space black `#050810` with subtle blue-purple gradient `#0d1b3e` on panels
> - Primary accent: Electric cyan `#00d4ff`
> - Secondary accent: Vivid violet `#7c3aed`
> - Text: Pure white `#ffffff` headings, `#94a3b8` body
> - Glassmorphism cards: semi-transparent panels with frosted glass effect and cyan border glow
> - NO clip art. NO stock-photo-style images. Use geometric shapes, icons, and abstract tech visuals.
> - Slide aspect ratio: 16:9 widescreen

## Animation Master Guide

Tell PowerPoint / the presentation tool to apply these animations:

### Transitions (between slides)
- Slide 1 → 2: **Split** (dramatic horizontal split)
- Slide 2 → 3: **Morph** (logo element transforms)
- Slide 3 → 4: **Wipe Right** (fast, cinematic)
- Slide 4 → 5: **Morph** (flow collapses to icon)
- Slide 5 → 6: **Fade through Black** (cinematic)
- Slide 6 → 7: **Zoom and Fade** (bar expands, resets)

### Element Animations (within slides)
- **Headlines**: Fade In from bottom, 0.3s, ease-out
- **Cards**: Fly In from bottom, staggered 0.4s delay per card
- **Flow diagram steps**: Wipe Right, 0.3s per step, sequential
- **Numbers counting up**: Custom animation effect (or use animated text)
- **Background neural network**: Continuous slow pulse loop (set as background GIF or animated shape)
- **Connecting lines in slide 3**: Draw (Wipe) animation left-to-right along each line
- **Skill cards in slide 5**: Custom spin/flip deal effect (or Fan effect)

### Timing
- Auto-advance: OFF (presenter controls timing)
- All animations: On Click or After Previous (presenter's choice)
- Suggest animation duration: 0.3–0.5s per element (snappy, not sluggish)
